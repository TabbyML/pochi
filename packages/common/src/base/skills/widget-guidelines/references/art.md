# Art Module

Use this module for illustration and generative art.

- Prefer SVG primitives: paths, circles, ellipses, polygons, repeated lines, and grouped transforms.
- Fill the canvas with intentional composition, but keep file size reasonable for streaming.
- Use hardcoded physical colors only for illustrative scenes where theme inversion would be wrong; otherwise use VSCode/Pochi variables.
- Animation is allowed only for local decorative or explanatory behavior, using CSS transform and opacity where possible.
